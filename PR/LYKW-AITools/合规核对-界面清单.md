# 合规核对 · 界面清单（可截图/可导出 PDF）

适用目录：`麟云开物-AI工具/LYKW-AITools`

## 如何保存“每个界面”的截图 / 文档

- **方式 A（推荐，最快）**：逐页截图
  - 打开 `index.html`
  - 左侧点页面（或用下方的 `#hash` 直接跳转）
  - Windows 截图：按 `Win + Shift + S` 选择区域（建议框住手机框/或平台端宽屏区域）→ 自动进剪贴板 → 粘贴到 Word/飞书文档/邮件即可

- **方式 B（整理成一个 PDF 文档）**：浏览器打印为 PDF
  - 打开 `index.html` → 切到要保存的页面
  - 按 `Ctrl + P` → 选择 **“另存为 PDF”**
  - 建议：每个页面单独导出 1 个 PDF，或统一导出到同一个文件夹再合并（飞书/Adobe/系统工具均可合并）

> 说明：本原型为单页多屏（hash 跳转）结构，目前未内置“一键批量截图导出”脚本；上述两种方式能稳定覆盖你的合规核对场景。

## 高敏界面（建议优先截图发商务/法务）

以下页面涉及 **分享引导 / 外站发布 / 奖励（券/红包）/ 资费套餐** 等表述：

- **用户端（C 端）**
  - `#c-pay-scan`：扫码支付页（券/优惠相关 UI）
  - `#c-pay-success`：支付成功页（“礼遇/抵用券”等引导）
  - `#c-promote`：推广中心（“随手分享”等引导 + 礼遇说明）
  - `#c-video`：发视频（“保存并分享 / 一键转发 / 已转发提交审核”等）
  - `#c-checkin`：点评打卡（生成文案配图 + “一键转发/提交审核”）
  - `#c-forward-proof`：上传转发凭证（“发布/转发截图作为凭证”）
  - `#c-reward`：奖励到账（抵用券到账、核销说明）

- **商家端（B 端）**
  - `#b-ai-promo`：产品宣传页（资费标准/套餐/价格/开通/补贴等）
  - `#b-todo`：用户分享列表（“确认发券/发放抵扣券”等）
  - `#b-todo-detail`：分享详情（核验凭证/发券按钮）
  - `#b-video-share-pool`：视频库·分享池（“加入分享菜单/平台审核/随机抽取”等）
  - `#b-share-copy-hub` / `#b-share-copy-work` / `#b-copy-library`：分享文案生成/文案库（平台投放/渠道配文）
  - `#b-promote-entry-config`：用户推广入口配置（控制外链/提示文案/WiFi 等）

- **平台端（电脑端）**
  - `#p-share-audit`：分享内容审核（敏感词/合规审核示意）
