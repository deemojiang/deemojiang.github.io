# 供应链平台 - 产品需求文档（PRD）

## 1. 项目概述

### 1.1 产品定位

B2B 供应链平台，连接供货商与零售商，支持商品批发、订单管理、对账结算等核心能力。

### 1.2 用户角色


| 角色   | 说明              | 使用端        |
| ---- | --------------- | ---------- |
| 供货商  | 上架商品、处理订单、发货、对账 | 供货商端（手机）   |
| 零售商  | 采购商品、下单、收货、管理门店 | 零售商端（手机）   |
| 平台运营 | 审核商户、管理商品/订单、客服 | 后台管理系统（PC） |


### 1.3 平台架构

- **供货商端**：移动端 H5/小程序/App
- **零售商端**：移动端 H5/小程序/App
- **后台管理系统**：PC Web

## 2. 供货商端功能需求

### 2.1 登录/注册（S-1）


| 功能点   | 描述                           |
| ----- | ---------------------------- |
| 手机号登录 | 输入手机号 + 验证码，支持登录/注册合一        |
| 服务协议  | 登录即表示同意服务协议与隐私政策             |
| 流程    | 登录成功后进入 S-2 资质认证（首次）或 S-3 首页 |


### 2.2 资质认证（S-2）


| 功能点   | 描述                   |
| ----- | -------------------- |
| 营业执照  | 上传营业执照图片，审核通过后才可上架商品 |
| 法人身份证 | 上传法人身份证正反面           |
| 提交认证  | 提交后等待平台审核            |
| 暂不认证  | 可跳过，先进入首页，后续在「我的」中补充 |
| 返回    | 可返回登录页               |


### 2.3 首页（S-3）


| 功能点        | 描述                |
| ---------- | ----------------- |
| 今日订单数/金额   | 展示今日订单数量及销售金额     |
| 待发货/待收款/售后 | 三类待办数量统计          |
| 快捷入口       | 发货、商品管理、订单管理，一键跳转 |


### 2.4 商品管理（S-4）


| 功能点     | 描述             |
| ------- | -------------- |
| 商品列表    | 展示所有商品，支持上架/下架 |
| 修改价格/库存 | 单商品改价、改库存      |
| 库存预警    | 库存不足时显示预警标识    |
| 批量改价    | 批量修改多个商品价格     |
| 批量改库存   | 批量修改多个商品库存     |
| 发布商品    | 跳转 S-5         |


### 2.5 发布商品（S-5）


| 功能点  | 描述                   |
| ---- | -------------------- |
| 商品标题 | 必填                   |
| 商品图片 | 支持多图                 |
| 规格   | 支持多规格（如 24瓶/件、500ml） |
| 供货价  | 单规格供货价               |
| 起订量  | 最小起订数量               |
| 库存   | 可售库存                 |
| 分类   | 食品/饮料/零食/日用品/冻品等     |


### 2.6 订单管理（S-6）


| 功能点  | 描述                 |
| ---- | ------------------ |
| 状态筛选 | 全部/待付款/待发货/已完成/已取消 |
| 订单列表 | 展示订单号、买家、金额、状态     |
| 一键发货 | 填写物流公司、运单号，确认发货    |
| 备注   | 订单备注               |
| 联系买家 | 联系零售商              |


### 2.7 订单详情/一键发货（S-7）


| 功能点  | 描述           |
| ---- | ------------ |
| 收货信息 | 收货人、地址、电话    |
| 商品明细 | 商品、数量、单价、小计  |
| 物流公司 | 选择快递公司       |
| 运单号  | 填写运单号        |
| 确认发货 | 提交后订单状态变为待收货 |


### 2.8 对账/结算（S-8）


| 功能点      | 描述                |
| -------- | ----------------- |
| 每日/每月销售额 | 按日/按月切换查看         |
| 订单明细     | 可查看每笔订单           |
| 应收金额     | 待结算金额             |
| 提现/结算记录  | 1.0 版本仅做记录，不做自动提现 |


### 2.9 我的/设置（S-9）


| 功能点     | 描述        |
| ------- | --------- |
| 资质认证    | 补充/修改资质信息 |
| 店铺信息    | 查看、编辑店铺资料 |
| 客服/联系平台 | 联系平台客服    |
| 修改密码    | 修改登录密码    |
| 退出登录    | 退出当前账号    |

## 4. 后台管理系统功能需求

### 4.1 数据概览（A-1）


| 功能点  | 描述      |

| 商户数量 | 供货商总数   |

### 4.4 商品管理（A-3）

## 6. 版本规划

### 6.1 1.0 版本（MVP）

- 供货商端、零售商端、后台管理全部核心功能
- 支付：微信支付 + 线下对账
- 提现：仅记录，不做自动提现
- 客服：基础会话能力

### 6.2 后续版本

- 自动提现/结算
- 更多支付方式
- 营销活动、优惠券
- 数据报表增强
