# 供应链平台（手机端）静态原型

## 如何查看

- 直接双击打开 `index.html` 即可（无需安装依赖、无需启动服务）。
- 如果你的浏览器阻止本地 `file://` 的某些行为，换用 Edge/Chrome 再试一次（本原型不依赖网络资源）。

## 原型包含页面

- 登录
- 首页（看板）
- 订单（含 Tab 过滤演示）
- 库存（含搜索与仓库切换演示）
- 物流追踪（含查询提示与时间线）
- 我的（个人信息与菜单）

## 小交互

- 底部导航切换页面（窄屏会进入单屏预览）
- 点击按钮会弹出提示 Toast（原型演示）
- 右上角可切换外观（仅影响外层背景与说明弹窗）
