## 一、产品概述

### 1.1 基本信息

| 项目 | 说明 |

| 状态 | 🔥 热卖、生效中 |
| 试用 | 支持免费试用、在线咨询 |

### 1.2 产品定位

聚合接待是一款专为**多店铺客服**设计的工具，面向电商多平台、多店铺的接待管理场景，核心解决店铺切换、响应延迟等问题，提升客服效率和客户体验。

---

## 二、痛点与价值

### 2.1 客服接待痛点

| 痛点 | 具体表现 |
|------|----------|
| 多店切换 | 同时登录多个店铺，频繁切换窗口到手软 |
| 消息遗漏 | 消息太多看花眼，总怕漏掉客户咨询 |
| 响应超时 | 响应超时频发，服务质量难保障 |
| 大促压力 | 大促爆单时，客服压力山大，接待不过来 |

### 2.2 产品价值

| 维度 | 效果 |
|------|------|
| 接单更顺 | 消息不遗漏，响应更及时 |
| 接待更快 | 快捷键+智能排序，效率翻倍 |
| 体验更优 | 客户满意，客服轻松 |

---

## 三、功能亮点

### 3.1 消息一屏聚合

- **九大平台聚合**：京东、淘宝、抖音、拼多多、快手、小红书、微信小店、得物、有赞
- **平台 Logo 标识**：点击即可跳转聊天，告别来回切换
- **集中展示待回复**：同步接待列表状态，待回复买家一目了然

### 3.2 智能接待策略

| 功能 | 说明 | 备注 |
|------|------|------|
| 场景信号灯 | 不同颜色提示消息优先级，一眼识别重要客户 | 需订购机器人 |
| 标星买家置顶 | 重点客户不错过 | - |
| 买家付款标签 | 实时展示「待付款」状态，售前促单更精准 | 需订购机器人 |

### 3.3 快捷键与灵活布局

| 功能 | 说明 |
|------|------|
| 快捷键切换 | 秒切未读消息、读秒最长会话，响应快人一步 |
| 吸附/置顶拖拽 | 窗口可吸附或置顶，始终保持在屏幕最前沿 |
| 自定义面板宽度 | 极简模式、全屏适配，界面可自定义 |
